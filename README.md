### Setup base for NestJS Project API

## Installation

```bash
$ git clone https://github.com/quitam/nestjs-setup-base.git
$ yarn install
# or
$ npm install
```

## Running the app

```bash
# development
$ yarn start
# or
$ npm run start

# watch mode
$ yarn dev
# or
$ npm run dev

# production mode
$ yarn start:prod
# or
$ npm run start:prod
```

## Test

```bash
# unit tests
$ yarn test
# or
$ npm run test

# e2e tests
$ yarn test:e2e
# or
$ npm run test:e2e

# test coverage
$ yarn test:cov
# or
$ npm run test:cov
```
